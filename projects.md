## Python та основи програмування

### [1_python_program](https://github.com/TeITp/1_python_program) — AI Greeter
Перша програма на Python: «розумне привітання» від ШІ, що запитує імʼя користувача, обирає випадкове привітання та виводить поточний час.
- **Бібліотеки:** `datetime`, `random`
- **Демонструє:** базовий ввід/вивід, робота з часом, випадкові числа.

### [Lab_3](https://github.com/TeITp/Lab_3) — Лабораторна №1: Основи Python
Знайомство з основними конструкціями мови Python: типи даних, змінні, цикли, розгалуження, обробка винятків та контекст-менеджери.
- **Технології:** Python

## Обʼєктно-орієнтоване програмування

### [Class_and_objects](https://github.com/TeITp/Class_and_objects) — Перша програма на ООП
Створення простих класів у Python: конструктор `__init__`, атрибути класу та обʼєкта, методи (instance, class, static), властивості.
- **Технології:** Python, Jupyter Notebook

### [5lab](https://github.com/TeITp/5lab) — Основи ООП у Python
Основні концепції ООП: класи, конструктор, методи, властивості (`@property`).
- **Технології:** Python, Jupyter Notebook

### [OOP_paradigms](https://github.com/TeITp/OOP_paradigms) — Ключові поняття ООП
Реалізація ключових принципів ООП на прикладі ігрової симуляції.
- **Технології:** Python
