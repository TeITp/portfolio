## Python та основи програмування

### [1_python_program](https://github.com/TeITp/1_python_program) — AI Greeter
Перша програма на Python: «розумне привітання» від ШІ, що запитує імʼя користувача, обирає випадкове привітання та виводить поточний час.

- **Теми:** `str`, `int`, `list`, `dict`, `set`, цикли `for`/`while`, `try`/`except`, `with`

## Обʼєктно-орієнтоване програмування

### [Class_and_objects](https://github.com/TeITp/Class_and_objects) — Перша програма на ООП
Створення простих класів у Python: конструктор `__init__`, атрибути класу та обʼєкта, методи (instance, class, static), властивості.
- **Технології:** Python 3, Jupyter Notebook
- **Теми:** конструктор, атрибути обʼєкта, instance/class/static методи, `@property`
- **Демонструє:** базові концепції ООП на прикладі класу `MyAnimals`.

### [5lab](https://github.com/TeITp/5lab) — Основи ООП у Python
Основні концепції ООП: класи, конструктор, методи, властивості (`@property`), змінні класу та магічні методи.
- **Технології:** Python 3, Jupyter Notebook
- **Теми:** `__init__`, `@property`, змінні класу, магічні методи (`__str__`, `__repr__` тощо)
- **Демонструє:** поглиблене вивчення ООП із практичними прикладами.

### [OOP_paradigms](https://github.com/TeITp/OOP_paradigms) — Ключові поняття ООП
Реалізація ключових принципів ООП на прикладі ігрової симуляції.
- **Технології:** Python 3
- **Парадигми:**
  - **Інкапсуляція:** клас `BankAccount` з приватними/публічними атрибутами та симуляцією банківських операцій.
  - **Наслідування:** класи `Vehicle` → `Car` з методом `start_engine`.
  - **Поліморфізм:** класи `Animal` → `Dog`, `Cat`, `Fish` з перевизначенням методу `speak()`.
  - **Абстракція:** абстрактний клас `Item` та класи зброї `Sword`, `Axe`, `Bow` з унікальними формулами атаки.
- **Демонструє:** покрокову гру з випадковим вибором зброї для гравця та комп'ютера.
