## Як створено цей сайт

1. Створено публічний репозиторій на GitHub.
2. У `Settings → Pages` обрано публікацію з гілки `main`, тека `/ (root)`.
3. До кореневої теки додано файл [`_config.yml`](_config.yml) з налаштуваннями:
    ```yml
    title: Portfolio Example
    description: Shows how to create portfolio for students.
    remote_theme: pages-themes/midnight@v0.2.0
    plugins:
      - jekyll-remote-theme
    ```
4. Як основу для сайту використано файл `README.md` та посилання на інші Markdown-сторінки.
